# AI・機械学習の学習記録：日本へのロードマップ

これは私の日々の進捗を記録するリポジトリです。私は現在、人工知能と機械学習を専攻するB.Tech2年生です。長期的な目標は、日本で働き、静かな生活を送ることです。

## 🚀 制作したプロジェクト・ツール
* **バイリンガルロードマップ:** キャリアパスとN5日本語学習目標をまとめたPythonスクリプト。
* **東京天気トラッカー:** 日本のリアルタイム気象データを取得するAPI連携ツール。
* **貯蓄予測ツール:** 線形代数の数式（$y = mx + b$）を使って日本移住の予算を予測する数学モデル。
* **単語マスター:** Pythonの辞書型とループを使った日本語N5単語の自動クイズツール。
* **安全予算管理ツール:** Try-Exceptブロックによるエラー処理とデータ検証を備えた堅牢な予算管理ツール。
* **目標日記:** PythonのI/O操作を使って学習記録を永続的に保存するファイル操作スクリプト。
* **単語カテゴライザー:** 論理的なクラスタリングで日本語単語を分類し、プロフェッショナルなドキュメント作成を練習するツール。
* **マントラカウンター:** AIのツリーアルゴリズムに不可欠な再帰処理（関数が自分自身を呼び出す）を練習するための瞑想ツール。
* **東京時刻同期ツール:** Pythonのdatetimeモジュールとタイムゾーン処理を練習しながら時差を計算するツール。
* **求人データフィルター:** 特定の技術・語学条件で候補者を絞り込む、MLデータ前処理を模倣したスクリプト。
* **ログ検証ツール:** `os`モジュールを使ってファイルパスの検証やファイルサイズなどのメタデータ処理を行うシステムチェックツール。
* **漢字デコーダー:** Pythonの辞書型を使って日本語文字と意味を対応させるルックアップツール。
* **東京レコメンダー:** ネストされた辞書型を使って複雑なデータ取得と条件分岐を練習する小さな推薦エンジン。
* **支出集計ツール:** 可変長引数`*args`を使って柔軟な関数を作る練習をするツール。
* **大学プロフィールビルダー:** AIモデルのパラメータ設定に不可欠なキーワード引数`**kwargs`を練習するツール。
* **日本都市ランカー:** Lambdaキーを使って複雑なタプル構造をソートするデータ整列ツール。AIランキングアルゴリズムに必須。
* **スキル重複排除ツール:** PythonのSetを使ってデータセットからユニークな特徴を抽出するユーティリティ。AIデータ前処理でよく使われる手法。
* **優先タスクセレクター:** 重み付き確率ロジックを実装して、AI/ML学習とN5日本語学習の優先度を数学的に決定するツール。
* **数字分類器:** ML基礎からディープラーニングへ移行し、TensorFlowで構築した数字分類器。
* **通貨コンバーター:** USD・EUR・GBP・INR・JPY間で金額を変換するPythonスクリプト。
* **デスクトップ通知ツール:** `plyer`ライブラリを使ってリマインダーのシステム通知を表示するスクリプト。
* **ウェブサイト状態チェッカー:** `requests`ライブラリを使ってウェブサイトの稼働状況とHTTPステータスコードを確認するツール。
* **スパイダーマン雑学クイズ:** 辞書ベースのロジックとユーザー入力処理を探求するインタラクティブなコマンドラインクイズゲーム。
* **NATOアルファベット変換器:** リスト内包表記を使ってユーザー入力をフォネティックコードに変換するスクリプト。
* **マイル→km GUIツール:** ラベル・入力フィールド・ボタンを使ったTkinterによる初めてのGUIアプリ。
* **ひな祭りオーガナイザー:** Pythonのクラスを使ったOOPベースのひな祭りスクリプト。
* **ホーリー色彩安全ツール:** ランダムな色を生成し、UIアクセシビリティのために明度を計算するスクリプト。
* **日本語単語ロガー:** ファイル操作（I/O）を使って日本語学習メモを保存・取得する入門スクリプト。
* **行列転置ツール:** リスト内包表記と線形代数の基礎を示す、行列を転置する数学スクリプト。
* **IoTセンサーモニター:** 温度アラートを外部ファイルに記録するIoTデータ監視シミュレーション。
* **キャリアキーワードスクレイパー:** 特定のキャリアキーワードやスキルでプロフィールをフィルタリングするAIベースのシミュレーションスクリプト。
* **SQLite3データベースマネージャー:** Pythonを使ってローカルSQLデータベースを作成・クエリする、RDBMSへの最初のステップ。
* **スタック（LIFO）シミュレーション:** Pythonのリストを使ってブラウザ履歴を模倣するスタックデータ構造の実装。
* **統計データアナライザー:** 平均・中央値・最頻値を計算してPythonで応用統計ロジックを練習するスクリプト。
* **RDBMS関係シミュレーター:** SQLの外部キーとテーブル結合を練習してRDBMSの概念を強化するPythonスクリプト。
* **ローカルポートスキャナー:** `socket`ライブラリを使ってホストの開いているTCPポートを確認するネットワークスクリプト。
* **円周率物理計算ツール:** `math.pi`定数を使って円運動の物理量を計算する物理ユーティリティスクリプト。
* **正規表現パターンスキャナー:** 正規表現を使ってテキストからメールアドレスや電話番号を抽出するテストマイニングスクリプト。
* **日本語フラッシュカードツール:** 辞書ベースのロジックを使ってN5試験のひらがなを練習する学習ツール。
* **DNS→IP リゾルバー:** `socket`ライブラリを使ってホスト名をIPアドレスに解決するネットワークスクリプト。
* **相関係数計算ツール:** ピアソン相関係数の公式を実装してデータの関係性を求める統計スクリプト。
* **データベース→CSVエクスポーター:** SQLレコードを取得してCSVファイルとして保存するデータポータビリティユーティリティ。
* **線形探索ビジュアライザー:** データ構造の練習として線形探索アルゴリズムをステップごとに可視化するツール。
* **移動平均計算ツール:** スライディングウィンドウ平均を使ってデータセットの「ノイズ」を平滑化するデータ処理スクリプト。
* **DB トランザクション シミュレーター:** SQL トランザクション（コミット・ロールバック）を実践し、RDBMS におけるデータの整合性を維持するスクリプト。
* **IoT ネットワーク チェッカー：** URLにpingを送信し、HTTPステータスコードとレスポンスタイムをファイルに記録するネットワークスクリプト。
* **キュー（FIFO）シミュレーション：** Pythonリストを使用してキューデータ構造を実装し、基本的なデータ構造のロジックを練習するプログラム。
* **マイルストーン トラッカー：** 50日間のコーディングを記念し、日本への旅を記録するための、OOPベースの特別プロジェクト。
* **NumPy 配列 基礎：** AIライブラリフェーズの開始。NumPyを使用して高速な数値配列と行列演算を実装。
* **線形方程式ソルバー：** NumPyのlinalgモジュールを使用して、4学期の数学における行列ベースの線形方程式を解くプログラム。
